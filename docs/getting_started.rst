Getting Started
===============

Welcome to the SOC Compass documentation! Getting started is easy.

The app currently runs via npm, so you will need to have Node.js installed. You can download it from https://nodejs.org/en/download/.

-----------------
Installing & Running SOC Compass
-----------------

```git clone https://github.com/soc-compass/soc-compass.git```
```cd soc-compass```
```npm install```
```npm run dev```
Visit `http://localhost:3000` to see the app.
